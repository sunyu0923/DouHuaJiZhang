# DouHuaJiZhang

豆花记账是一款 iOS 记账应用，配套 Go 后端服务，支持用户登录、家庭共享账本、交易记录、攒钱计划、投资管理和健康记录。

## 主要模块

- **认证**: 手机号密码/验证码登录、注册、刷新令牌。
- **账本**: 个人/家庭账本创建、成员管理、权限校验。
- **交易**: 按账本记录收入/支出，支持统计和日历视图。
- **攒钱计划**: 设置月度/年度目标并按交易数据计算进度。
- **投资管理**: 记录用户投资产品和行情入口。
- **健康记录**: 拉屎记录、月经记录与预测。

## 构建与测试

```bash
./run_tests.sh go
```

也可以进入后端目录单独运行：

```bash
cd server
go test ./... -count=1
```
